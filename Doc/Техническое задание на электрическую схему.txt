Техническое задание для электрической схемы ПЛК.

Дискреные входы транзисторные гальванически развязаннные 24VDC 6mA PNP/NPN.
Дискреные выходы транзисторные гальванически развязаннные 24VDC 500mA PNP/NPN.
Дискретные входы должны быть совместимы с 2х проводными датчиками IFM ??? и им подобными.
Предпочтительно использовать PNP входы / выходы с общим минусом питания.
Скоростные версии дискретных входов / выходов должны иметь скорость 100KHz.
Аналоговые входы гальванически развязаннные 0(4)...20mA / 0...10VDC 16Bit 100ms.
Аналоговые выходы гальванически развязаннные 0(4)...20mA / 0...10VDC 16Bit 100ms.
Напряжение на разомкнутой токовой петле аналогово выхода не менее 23VDC.
Клеммы для подключения внешних проводов пружинные 0,5...0,75мм2.
Расположение клемм вертикальное.

Рекомендуемое количество ресурсов базовой комплектации PLC.
8 digital input 24VDC (4Di HTL encoder)
8 digital output 24VDC
4 analog input 4-20mA
2 analog output 4-20mA
1 RS485 MODBUS RTU
1 Ehernet MODBUS TCP / NTP CLIENT
Дополнительно:
SWD \ JTAG + RTC + 1Kb EEPROM + 1Kb NVRAM

Размеры рекомендуемых к использованию SMD компонентов
Минимально возможный для ручной пайки 0603
Самых ходовой 0805
Для мощных применений 1206
Точность резисторов 1%

Сеть RS485:

Рассмотреть возможность перехода на ADM2483BRWZ тк дешевле и абонентов больше.
Рассмотреть возможность установки зеленого светодиода мигающего при обмене данными.
